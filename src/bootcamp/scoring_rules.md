# Hero Terminal – Scoring Guidelines (v1)

- Beginner missions: 25–75 points
- Intermediate missions: 75–150 points
- Advanced missions: 150–250 points

## Rules

- Never assign more than 250 points to a single mission.
- Never go below 10 points.
- For bootcamps with 3–5 missions, total score should land between ~200 and 600 points.
- If you think a mission deserves more than 250 points, split it into multiple missions.
